StochSS: Stochastic Simulation as a Service

Advisors: Prof. Chandra Krintz, Prof. Rich Wolski
RACE Lab, UC Santa Barbara

StochSS is a NIH funded project whose aim is to provide an integrated development environment (IDE) for running discrete stochastic simulations of biochemical networks and population systems in cloud somputing environments.

  • Built a generic cloud computing framework for configuring virtual machines and auto-deploying/managing arbitrary scientific simulation programs in the cloud by wrapping the source code as a web service.
  • Developed Flex Cloud, a lightweight cloud service abstraction layer for supporting simulation runs over different infrastructures (physical, virtual, as well as public or private clouds).
Terrier.org

Terrier IR: Query Expansion Improvement

Advisors: Prof. Mandar Mitra
CV & PR Unit, Indian Statistical Institute

Terrier is a highly flexible, efficient, and effective open source search engine, readily deployable on large corpora of documents.

  • Optimized search in Terrier IR engine by improving query expansion results, exploiting semantic relationships amongst words in conjunction with Local Context Analysis Techniques and implementing them in its Divergence From Randomness (DFR) framework.
  • A part of the project, integration of WordNet (taxonomy of synsets) in Terrier was done as my Summer Project.
